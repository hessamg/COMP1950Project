The use of HTML <blockquote>tag.

The <blockquote> tag specifies a section that is quoted from another source.To use <blockquote>,it must contain only other block-level elements.

Example:
<blockquote>
<p>This is a long quotation.This is a long quotation </p>
<blockquote/>
Note:Use <q> tag for short quote,for example:
<q>“What separates design from art is that design is meant to be… functional.” ― Cameron Moll </q>
“What separates design from art is that design is meant to be… functional.” ― Cameron Moll

The use of HTML <select> tag.

The <select> element is used to create a drop-down list. It contains the <option>tag inside to define the available option in the list.

Example:
<select>
<option value="name">Name </option>
<option value="name">Gender </option>
<option value="name">Birthdate </option>
</select>
